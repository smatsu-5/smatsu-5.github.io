Vegetable Weight Progressive Web App

The PWA was created to aid the produce department in weighting their products according to a target cost. The app allows users to view a produce's price/lbs and weights for the target costs. The app is hosted in Google Firebase and uses its build components to store data and images.
Via a passcode, a manager can change the data for the produce and add new items to the database.

Technologies used: Google Firebase, HTML, CSS, Javascript, Jquery, Bootstrap
