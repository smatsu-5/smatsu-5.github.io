    Vegetable Price Web App

    Previously spreadsheets were used to determine pricing for our products. The spreadsheet was a bit messy so this web app simplifies with a simple UI. No need for multiple rows of unnecessary data for the user.

    The app leverages Google Firebase for both hosting and real-time database functionality, allowing users to view and update product pricing data instantly.

    This solution has simplified the process and made has reduced errors by the user compared to if they used a spreadsheet.

    Vegetable Weights PWA

    Developed a web app hosted by Google Firebase to efficiently make price-based weight calculations for the produce department. The user enters the current price per pound of the product and the app determines the recommended weights to meet the specified target costs.

    A manager feature is included where managers can adjust pricing and target costs. They are also a simple UI that allows managers to create and remove products from the lineup. The app uses Firebase for hosting and database services to store data and images.
